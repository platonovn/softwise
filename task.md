Необходимо написать приложение, состоящие из двух страниц

На первой странице просто ссылка перехода на другую страницу, на второй странице есть форма, состоящая из трех полей

- date (просто input[type="date"]), 
- first_name (строка), 
- last_name (строка)

и кнопка отправить. Необходимо постом отправить json с данными на `https://haff-273414.appspot.com`, в ответ придет структура
`{success: frue, data: [...]}` если все хорошо, и `{success: false, error: {...}}` с кодом 400 если есть ошибки, в `error` при этом будет содержаться объект 
вида `{field_name: [err_text]}` для каждого филда, где есть ошибки. 

Если пришел код 200, то необходимо вывести на странице содержание data и сериализовать форму в адрес страницы. Если 400, то показать в каких полях есть ошибки.

Пока данные грузятся рисовать спиннер. При отрытии страницы, если в адресной строке есть форма, необходимо восстановить форму из данных адресной строки и подтянуть данные, в идеале тянуть 
данные прямо при навигации (использую Route Resolvers), что бы при открытии страницы по ссылке спиннера загрузки данных не было.     

Если пришел 500, то можно просто показать алерт о том, что с сервером что-то не так.

Например запрос

``
curl -sv -XPOST -H 'Content-Type: application/json' -d '{"first_name": "Ivan", "last_name": "Ivanov", "date": "2019-01-02"}' https://haff-273414.appspot.com
``

может вернуть как успешный статус

        {
         "success" : true,
         "data" : [
            {
               "date" : "2019-01-02",
               "name" : "Ivan Ivanov"
            },
            {
               "name" : "Ivan Ivanov",
               "date" : "2019-01-02"
            }
         ]
        }


так и ошибку (в 50% случаев будет ошибка)

    {
      "success": false
      "error": {
        "first_name": [
          "'Ivan' is not allowed"
        ]
      },
    }


Если в `first_name` есть пробел, то ошибка будет всегда. Например:

``
curl -sv -XPOST -H 'Content-Type: application/json' -d '{"first_name": "Ivan Ivanov", "last_name": "Ivanov", "date": "2019-01-02"}' https://haff-273414.appspot.com
``

Возвращает

      {
         "success" : false,
         "error" : {
            "first_name" : [
               "No whitespace in first name is allowed"
            ]
         }
      }
